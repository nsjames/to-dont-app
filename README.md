# TO-DON'T App

This is a simple app to help you to not do things that you don't want to do.

It's built using:

- React / Next 13
- Solidity
- Hardhat
- Ethers
- TailwindCSS

To run this app:
```bash
cd web
npm run dev
```

If you want to see how to build this, head over to: https://youtu.be/bbSLJy-7tm0
